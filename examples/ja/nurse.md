# 病院ナース勤務表づくり（月次ロスター・30日）  
  
## 1. シナリオ  
あなたは一般病棟（急性期寄り）の勤務表担当です。来月30日間のロスターを作成します。  
患者数は平日が多め、週末は少なめですが、月の中盤と月末に入退院の波が来ます。  
看護師の「お願い（希望休・やりたい/避けたい勤務）」と、病棟の「安全に回すための現場ルール」の両方を踏まえて、無理のないシフト表を作ってください。  
  
## 2. 前提（病棟・期間・シフト）  
- **病棟**：一般病棟（単一病棟）  
- **期間**：30日（Day1〜Day30）  
- 便宜上、**Day1＝月曜日**とします（Day7＝日曜、以降繰り返し）  
- **シフト**：1日3交代  
- **早番 (M)** 08:00–16:00  
- **遅番 (E)** 16:00–24:00  
- **夜勤 (N)** 24:00–翌08:00（明けは原則お休み）  
  
## 3. 病棟のルール（用語は平易に）  
### 3.1 絶対に守るルール  
1. **1人1日1シフト**（掛け持ちなし）。  
2. **各シフトは最低2名で入る**（一人現場は作らない）。  
3. **各シフトに「有資格者」（リーダーまたは正看）を少なくとも1名**。  
4. **夜勤の次の朝は勤務を入れない**（体調・安全配慮）。  
5. **連続勤務は最長6日**（7日目は必ず休みを入れる）。  
6. **事前に決まっている不在日は入れない**（研修・外来応援・私用など）。  
  
### 3.2 できれば守りたいルール（優先度の相談余地あり）  
1. **必要人数に近づける**（下表の「目安人数」）。  
2. **各人の夜勤回数は偏らせない**（目安回数を超えないように）。  
3. **週末はできるだけ公平に**（同じ人に偏らせない）。  
4. **希望休・やりたい/避けたい勤務をできる範囲で尊重**。  
5. **リーダーは毎シフト1名いると安心**（どうしても足りない日は正看で代替）。  
  
> ※「できれば守りたい」は、全体のバランスや安全を優先しつつ、現場と相談して調整して構いません。  
  
## 4. 1日の必要人数の目安  
- **平日（Mon–Fri）**：M=4, E=4, N=3  
- **週末（Sat, Sun）**：M=3, E=3, N=2  
- **特記事項（増員/減員の目安）**  
- **Day10（木）**：入院増加見込み → M+1, E+1（例：M=5, E=5）  
- **Day18–Day19（土日）**：病棟イベント対応 → N+1（例：N=3）  
- **Day29（月）**：定期点検で一部病室クローズ → 全シフト−1（ただし最低2名は確保）  
  
## 5. 人員一覧（役割・勤務の目安・不在予定）  
- **役割の呼び方**  
- **リーダー**：経験豊富、緊急対応や指示ができる  
- **正看**：通常業務を自立して担当できる  
- **補助**：ケアや介助を主に担当（単独夜勤は不可／必ず有資格者と一緒）  
  
- **勤務回数の目安（30日）**  
- **常勤**：おおむね20シフト前後  
- **非常勤**：おおむね14シフト前後  
- **夜勤回数の目安（上限イメージ）**：  
- リーダー：**〜6回**／月  
- 正看：**〜5回**／月  
- 補助：**夜勤は可（必ず有資格者と同じシフト）**／回数は相談

## 7. あなたへの依頼  
**次の条件を満たす「30日分の勤務表」を作ってください。**  
  
1. 上記の**絶対に守るルール**を満たす。  
2. **必要人数の目安**に、なるべく近づける（特記事項の日は増減あり）。  
3. **夜勤・週末・勤務回数**が特定の人に偏りすぎないように分散する。  
4. **不在日**と**希望メモ**をできる範囲で反映する（難しい場合はどこをどう譲ったか分かるように）。  
5. **月末の人手不足**（L003不在）と**入院増・イベント日**への対応方針を、簡単なメモで添える。  
  
> 仕上がりは、例えば「Day×シフト（M/E/N）×担当者ID」の表、あるいは日別の一覧など、**誰が見ても現場で運用しやすい形**であればOKです。  
> 途中で難所が出たら、**どのお願いを優先し、どこを入れ替えたか**をメモに残してください（現場での合意形成がしやすくなります）。
