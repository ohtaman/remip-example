# 📋 物流センターのピッキング担当 勤務表づくり

来月30日間の物流センター（単一拠点）のシフト表を作成して欲しい。ECの注文量は平日が多め、週末は少なめですが、月の前半にセール、月末に棚卸しがあり波があります。
スタッフの「お願い（希望休・やりたい/避けたい勤務）」と、現場の「安全に回すための運用ルール」の両方を踏まえて、無理のない勤務表を作ってください。

## 前提（拠点・期間・シフト）

- **拠点**：単一物流センター（1フロア）
- **期間**：30日（Day1〜Day30）
- 便宜上、**Day1＝月曜日**とします（Day7＝日曜、以降繰り返し）
- **シフト**：1日3交代（8時間×3）
- **早番 (M)** 06:00–14:00
- **中番 (D)** 14:00–22:00
- **夜番 (N)** 22:00–翌06:00（明けは原則お休み）

## 現場のルール

### 絶対に守るルール

1. **各シフトは最低4名で稼働**（安全・品質のため）。
2. **各シフトに「フォークリフト有資格者」を少なくとも1名**（入出庫・補充対応）。
3. **各シフトに「リーダー」を少なくとも1名**（指示出し・当日判断）。
4. **夜番の次の日は勤務を入れない**（連続深夜回避）。
5. **連続勤務は最長6日**（7日目は必ず休み）。
6. **事前に決まっている不在日は入れない**（研修・私用など）。
7. **未経験者（新人）は夜番に入れない**（安全上の制約）。

### できれば守りたいルール

全体の安定稼働を優先しつつ、現場と相談して調整します。

1. **夜番回数は偏らせない**（上限・目安を超えないように）。
2. **週末勤務はできるだけ公平に**（同じ人に偏らせない）。
3. **希望休・やりたい/避けたい勤務をできる範囲で尊重**。
4. **リーダーはできれば毎シフト2名いると安心**（繁忙日は特に）。

## 1日の必要人数（最低配置）

- **平日（Mon–Fri）**：M=7, D=7, N=6
- **週末（Sat, Sun）**：M=6, D=6, N=5

### 必要人数についての特記事項（波への対応）

- **Day5（金）〜Day7（日）**：月初セール出荷増 → **M+2, D+2**（例：平日ならM=9, D=9）
- **Day14（月）**：大型入荷で補充多め → **N+2**（例：N=8）
- **Day23（水）〜Day24（木）**：返品増 → **D+2**（例：D=9）
- **Day29（月）**：棚卸しでピッキング縮小 → **全シフト−2（ただし最低4名は確保）**
- **Day30（火）**：棚卸し後の出荷戻り → **M+1, D+1**

## 人員一覧

- **役割の呼び方**
- **リーダー**：当日運用の判断、品質/安全の最終責任
- **フォーク有資格者**：フォークリフト免許あり（補充・入出庫対応）
- **一般スタッフ**：ピッキング/梱包中心
- **新人**：入社1ヶ月以内（夜番不可、重作業は補助）

- **勤務回数の目安（30日）**
- **常勤**：おおむね20シフト前後
- **非常勤**：おおむね12〜14シフト前後
- **夜番回数の目安（上限イメージ）**
- リーダー：**〜7回／月**
- フォーク有資格者：**〜8回／月**
- 一般スタッフ：**〜6回／月**
- 新人：**夜番不可**

## 依頼内容

上記の条件を満たす30日分の勤務表を作ってください。

### 注意事項

1. 出力形式は縦にスタッフID、横に日付を並べたマークダウン形式のテーブルとすること（セルには **M / D / N / O(休)** を入れる）。
2. 満たせなかったルールや考慮点について、**絶対に守るルール**と**できれば守りたいルール**それぞれについて整理して箇条書きで出力すること。
